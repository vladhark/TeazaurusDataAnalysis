# Поиск похожих предложений (semantic similarity search)

## определение

Задача нахождения в корпусе предложений, наиболее близких по смыслу к заданному запросу. Используется косинусное сходство между векторами предложений (полученными суммированием эмбеддингов токенов). Полезна для анализа текстовых данных, рекомендаций, кластеризации.
