# Косинусное сходство (cosine similarity)

## определение

Мера схожести двух векторов, определяемая как косинус угла между ними. Формула: cos(θ) = (A·B) / (||A||·||B||). Используется для поиска ближайших векторов, в том числе для поиска похожих предложений (sklearn.metrics.pairwise.cosine_similarity).
